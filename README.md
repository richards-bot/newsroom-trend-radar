# Newsroom Trend Radar

A lightweight dashboard for tracking trends relevant to The Guardian's video-first and audience growth strategy.

## Includes
- MAM/workflow signals (iconik, LucidLink, Backlight)
- Video distribution trends
- AI automation opportunities
- Suggested experiments
- Daily source links

## Deploy (free)
This repo is configured for GitHub Pages via Actions.

After push:
1. Open repo Settings → Pages
2. Ensure Source is **GitHub Actions**
3. Wait for workflow completion

Live site URL should be:
`https://richards-bot.github.io/newsroom-trend-radar/`
